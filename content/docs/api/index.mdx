description: API Reference for Propeller
---

The Propeller API is documented using the [OpenAPI 3.0](https://swagger.io/specification/) specification. The full spec is available at [`openapi.yaml`](/openapi.yaml) and covers all Manager HTTP endpoints including task management, job scheduling, workflow orchestration, and federated learning.

A [Postman collection](/postman_collection.json) is also available for interactive API exploration.
