This is a simple example using the [Elastic HAL](https://github.com/elasticproject-eu/wasmhal) `elastic:attestation` interface, introduced in [PR #4](https://github.com/elasticproject-eu/wasmhal/pull/4). It demonstrates the intended nonce-based attestation API via the `elastic:attestation/hal` interface, calling `get-attestation` with a 16-byte nonce to request an attestation report from the underlying platform.

> **Note:** The `elastic:attestation/hal` interface from PR #4 is not yet implemented in the proplet runtime. Running this example will fail at instantiation with an unknown import error. This example serves as a placeholder for when the interface lands.

## Build

```bash
cd propeller
make attestation-test
```

Your output should look like this:

```bash
cd examples/attestation-test && cargo build --release
    Finished `release` profile [optimized] target(s) in 0.01s
cp examples/attestation-test/target/wasm32-wasip1/release/attestation-test.wasm build/attestation-test.wasm
```
