This is a WASI NN (Neural Network) example. It demonstrates running machine learning inference using the WASI NN API with an OpenVINO backend.

For a comprehensive guide on WASI-NN concepts, supported backends, and detailed build instructions, see the [WASI-NN](/docs/wasi-nn) documentation.

## Build Wasm

Build the WASI NN example to WebAssembly:

```bash
cd propeller/examples/wasi-nn
cargo build --target wasm32-wasip1 --release
```

Your output should look like this:

```bash
Compiling wasi-nn v0.1.0
Compiling wasi-nn-example v0.1.0 (/home/user/propeller/examples/wasi-nn)
 Finished `release` profile [optimized] target(s) in 2.34s
```

The compiled WASM file will be at `target/wasm32-wasip1/release/wasi-nn-example.wasm`.

## Prepare Model Files

The example requires OpenVINO model files in a `fixture` directory:

```bash
mkdir -p fixture
# Download or copy your OpenVINO model files:
# - model.xml (graph definition)
# - model.bin (weights)
# - tensor.bgr (input tensor in BGR format, 224x224x3)
```

```bash

-H "Content-Type: application/json" \
-d '{"name": "_start"}'
```


## Running with WasmTime

You can also run the example directly with WasmTime (requires OpenVINO backend):

```bash
wasmtime run --dir fixture \
  --wasi-modules=experimental-wasi-nn \
  target/wasm32-wasip1/release/wasi-nn-example.wasm
```

Your output should look like this:

```bash
Read graph XML, first 50 characters: <?xml version="1.0" ?>
<net batch="1" name="mob
Read graph weights, size in bytes: 3469612
Loaded graph into wasi-nn with ID: 0
Created wasi-nn execution context with ID: 0
Read input tensor, size in bytes: 602112

Found results, sorted top 5: [InferenceResult(281, 0.9875), InferenceResult(282, 0.0078), InferenceResult(285, 0.0021), InferenceResult(287, 0.0009), InferenceResult(290, 0.0004)]
```

The numbers represent ImageNet class IDs and their confidence scores.
